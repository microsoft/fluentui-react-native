# Overall plan

This package will be a new component library of components that will be generated by agents.

- Package name: @fluentui-react-native/agentic-components

## Structure

The package should have a rough structure as follows:

- /src - root of the source directory
- /src/components - root of component definitions
- /src/components/button - simple react-native button implementation just to start
- /src/components/button/button.tsx - button implementation
- /src/components/button/button.test.ts - button jest tests
- /src/components/button/button.stories.ts - button stories for storybook
- /storybook - storybook app that will run any stories in the project

## Tasks

1. Create the rough package shape with the configs, package.json, and directories with stub files
2. The package should target react-native-macos right now, the same version that the rest of the repository uses
3. Create a simple Button control in the directory above just to scaffold tests and storybook integration
4. Create a storybook react-native test app using react-native-macos in the ./storybook directory which loads stories from the library
